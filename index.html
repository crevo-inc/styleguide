スタイルガイド

Layouts新規作成するには、以下のテンプレートを読み込み、
必要に応じてスタイルガイドの各モジュールを参照してください。

  • Front Page

    Front Pageのテンプレートです。
    以下の「詳細」からレイアウトをできます。
    ※スマホサイズデータはspフォルダ内にあります。

    詳細
    ダウンロード

Base

jQuery

<body>要素の終了タグ手間に読み込む。bootstrapを使用するため、bootstrap.min.jsの前に設置。

  • HTML

    <script src="https://ajax.googleapis.com/ajax/libs/jquery/1.11.3/jquery.min.js"></script>

Bootstrap

Crevo全体のレイアウト／スタイル／スクリプトの標準機能として使用。
<head>要素内にbootstrap.min.css>bootstrap.theme.cssの順で読み込む。
bootstrap.min.jsは<body>要素の終了タグ手間のjquery.min.jsの後に読み込む。
bootstrapデータ自体は編集せず、後に読み込むスタイルシートに編集コードを記述し、上書きする。

  • HTML

    <!-- bootstrap css -->

    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css" integrity="sha512-dTfge/zgoMYpP7QbHy4gWMEGsbsdZeCXz7irItjcC3sPUFtf0kuFbDz/ixG7ArTxmDjLXDmezHubeNikyKGVyQ==" crossorigin="anonymous">

    <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap-theme.min.css" integrity="sha384-aUGj/X2zp5rLCbBxumKTCw2Z50WgIr1vs/PFN4praOTvYXWlVyh2UtNUU0KAUhAX" crossorigin="anonymous">

    <!-- bootstrap js -->

    <script src="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/js/bootstrap.min.js" integrity="sha512-K1qjQ+NcF2TYO/eI3M6v8EiNYZfA95pQumfvcVrTHtwQVDG+aHRqLi/ETn2uB+1JqwYqVG3LIvdm9lj6imS/pQ==" crossorigin="anonymous"></script>

Viewport

端末やブラウザ毎に幅を自動調整させるために、幅を取得し、モバイル端末でのズーム倍数初期値を指定（ズーム可）。<head>要素内に記述。

  • HTML

    <meta name="viewport" content="width=device-width, initial-scale=1">

Grids

Container

media queryで端末毎に幅が切り替わる際の、固定のコンテンツ表示領域を指定。
各エリアのコンテンツを.containerタグで括る。
PC表示で背景幅100%指定の場合、.container指定要素は背景指定タグの内側で指定。
コンテンツを常時画面幅100%に表示する場合は.container-fluidを使用。

  • HTML

    <div class="container">
        ...
    </div>

    <div class="container-fluid">
        ...
    </div>

Media Queries

端末ごとにコンテンツ表示幅の切り替わるポイントを指定。
例として@media (max-width: @screen-xs-max) {}の{}にコードを記述した場合、スマートフォン表示サイズのみに対応。
※モバイルファーストで記述する場合、media queryの記述する順が逆転するので注意。

  • HTML


    @media (min-width: @screen-lg-min) { ... }
    @media (min-width: @screen-md-min) { ... }
    @media (min-width: @screen-sm-min) { ... }
    @media (max-width: @screen-xs-max) { ... }

    /* モバイルファーストの場合 */

    /* Extra small devices (phones, less than 768px) */
    /* No media query since this is the default in Bootstrap */

    /* Small devices (tablets, 768px and up) */
    @media (min-width: @screen-sm-min) { ... }

    /* Medium devices (desktops, 992px and up) */
    @media (min-width: @screen-md-min) { ... }

    /* Large devices (large desktops, 1200px and up) */
    @media (min-width: @screen-lg-min) { ... }

Pagination

コンテンツ量の多い一覧ページなど、ページ送りが必要な箇所に使用。

  • « First
  • ‹ Prev
  • 1
  • 2
  • 3
  • …
  • Next ›
  • Last »
  • HTML
  • CSS

    <ul class="pagination">
        <li class="disabled"><span>« First</span></li>
        <li class="disabled"><span>‹ Prev</span></li>
        <li class="active"><a href="#">1</a></li>
        <li class=""><a rel="next" href="/projects?page=2">2</a></li>
        <li class=""><a href="/projects?page=3">3</a></li>
        <li class="disabled"><a href="#">…</a></li>
        <li><span><a rel="next" href="/projects?page=2">Next ›</a></span></li>
        <li><span><a href="/projects?page=13">Last »</a></span></li>
    </ul>



    .pagination {
        margin: 0;
    }
    .pagination>.active>a,
    .pagination>.active>a:focus,
    .pagination>.active>a:hover,
    .pagination>.active>span,
    .pagination>.active>span:focus,
    .pagination>.active>span:hover {
        background: #ed4e52;
        border-color: #ed4e52;
        color: #ffffff;
    }
    .pagination>li>a,
    .pagination>li>span,
    .pagination>li>span>a {
        color: #333333;
    }

Pankuzu

入れ子式に配置されたページの現在地を表すのに使用。最後のテキストのみリンクを外す。

  • text-01
  • text-02
  • text-03
  • HTML
  • CSS

    <ul class="pankuzu">
        <li><a href="#">text-01</a></li>
        <li><a href="#">text-02</a></li>
        <li>text-03</li>
    </ul>



    .pankuzu {
        overflow: hidden;
        list-style-type: none;
        padding: 0;
        margin: 0;
    }
    .pankuzu li {
        float: left;
        color: #333333;
    }
    .pankuzu li:after {
        content: ">";
        margin-left: 10px;
        margin-right: 10px;
    }
    .pankuzu li:last-child:after {
        content: "";
    }
    .pankuzu li a {
        color: #333333;
        text-decoration: underline;
    }
